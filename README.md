# Restaurant Listing Website

This is a simple restaurant listing website built using **HTML, CSS, and JavaScript**.  
It displays multiple restaurants as cards and allows users to filter and sort them based on rating, price, distance, and alcohol availability.

---

## Features

- Display restaurants with images, name, rating, location, and cuisine
- Sticky filter bar that stays at the top while scrolling
- Filter options:
  - Rating 4.5+
  - Serves Alcohol
  - Sort by price (High to Low / Low to High)
  - Sort by distance
- Popup filter menu
- Scroll-to-top button
- Collection section with left/right slider

---

## How to Use

1. Download or clone this repository
2. Add your restaurant data in the `restraunts` array inside `script.js`
3. Add restaurant images in the `Images` folder
4. Open `index.html` in your browser

No backend or server setup is required.
